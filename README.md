# MSTTS TXT在线听书文字转语音阅读器
演示：https://www.bilibili.com/video/BV1WZrPBsEjw/
## 使用方法

**导入TXT，鼠标拖动选取文字开始位置，然后点击播放**

AI写的所以有些毛病，请多包涵，最大问题是句子的停顿问题。

## 更新
添加存档读档功能，存档会存储在服务，存档是存储当前播放到的句子。
